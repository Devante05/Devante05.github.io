Weather Dashboard is a weather app I created to give real time weather data as well as a five day forcast for any city in the world! This application uses 3rd party APIs to retrieve data about temperatures, wind speed, UV index, humidity as well as an illustation describing the current weather. This web application was created using Javascript and Jquery. Bootstrap was used for styling.
